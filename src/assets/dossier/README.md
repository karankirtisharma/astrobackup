# Dossier cutout art

Drop transparent-background images here and they replace the procedural SVG
art in the dossier panels automatically — no code change. Resolution happens
at build time (`import.meta.glob` in `src/ui/dossier/Card.tsx`), so a missing
file costs nothing at runtime; it simply renders the SVG fallback.

| Filename           | Module                        | Suggested subject                          |
| ------------------ | ----------------------------- | ------------------------------------------ |
| `anatomy.*`        | CYPHERPUNK → BIO METRICS      | Male anatomical/muscular figure, 3/4 view   |
| `fingerprint.*`    | CYPHERPUNK → DATA FOOTPRINT   | Fingerprint, high contrast                  |
| `mri-coronal.*`    | CYPHERPUNK → NEURAL MAP       | Coronal MRI brain slice                     |
| `mri-axial.*`      | CYPHERPUNK → NEURAL MAP       | Axial MRI brain slice                       |
| `radar.*`          | CYPHERPUNK → CODENAME         | Radar/reticle graphic                       |
| `signature.*`      | CYPHERPUNK → CODENAME         | Handwritten signature stroke                |
| `skull.*`          | ASTRONAUT → VITALS            | Cranial x-ray, front view                   |
| `planet.*`         | ASTRONAUT → MISSION PROFILE   | Cratered sphere / moon                      |
| `suit.*`           | ASTRONAUT → SUIT DIAGNOSTICS  | Full-body space suit, standing              |
| `suit-xray.*`      | ASTRONAUT → X-RAY SCAN        | Same suit as wireframe/x-ray                |

Accepted extensions: `.png`, `.webp`, `.jpg`. **Alpha is required** — the
panels are glass (translucent + `backdrop-filter`), so an image with a baked
black background shows as a solid rectangle floating on the blur.

Most files here are generated by `npm run art:extract`, which crops them out
of the reference boards in `assets-src/boards/` and keys them: luminance
becomes the alpha channel and RGB is flattened to white. That is
un-premultiplied, so compositing over black reproduces the source exactly
while over glass it reads as a lit HUD overlay. Tune the crop boxes in
`scripts/extract-dossier-art.mjs`.

Hand-authored replacements just need a genuine transparent background.
Do **not** reach for `mix-blend-mode: screen` to fake it — it composites
against whatever shows through the glass, which lifts the black field back
into view.

Images are decorative (`aria-hidden`) — every module states its data in text,
so nothing is lost when the SVG fallback is used.
